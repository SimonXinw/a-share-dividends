说明：

  • 双击单元格即可编辑（股票代码、股名、行业、去年股息率、预估每股分红 由计算/数据源给出，不可改）。
  • 编辑保存后会立即重新计算「去年股息率」「今年预估每股分红」「今年预估股息率」。
  • 计算公式：今年预估净利润 = 去年净利润 - 去年「已替换季度」净利润 + 今年已发布季度净利润；今年预估每股分红 = 去年每股分红 × 今年预估净利润 / 去年净利润；今年预估股息率 = 今年预估每股分红 / 当前股价。
  • “当前市值/去年年末市值”在股价同步阶段一并更新，所以点「一键同步全部」即可。
  • 修改字段后会保存到 a_share_overrides 表，不污染原始抓取数据；点「恢复原值」可清除覆盖。
  • 股息率配色（A 股传统：红涨绿跌）： 达标线 {{ thresholdLabel }} = 中国 10 年期国债 {{ riskFreeLabel }} + 3% 股权风险溢价（自动跟随宏观利率）。 低于达标线 浅绿 绿 深绿 墨绿 越远越深； 高于达标线 浅橙红 红 深红 墨红 越远越显眼； ≥ 9% 警示「高息陷阱」（业绩暴雷 / 一次性特别分红 / 数据未除权），需自行核实。
  • 趋势箭头（同 K 线惯例）： ↑ 预估股息率高于去年（向好，红）； ↓ 低于去年（转差，绿）； ─ 基本持平。
  • 行业基准：「去年股息率」单元格右侧的 +x.xx% 表示高于同行业均值（红，强）； -x.xx% 表示低于均值（绿，弱）。鼠标悬停可见行业均值。
